About this app

Calculator with 5-Key Switch Interface

  • Instructions for using the app

    • W: Up, A: Left, S: Down, D: Right, P: Select

  • Test Results

    • I tested the input time of the following equations using one hand vs two hands:

    ((5*3)/2)+9

    ((40/4)+12)-10

    ((25+10)-15)*4

    ((0-10)/5)*12

    ((9*5)-4)+3)/2

    One Hand or Two: Time: Errors:
    Two Hands 1:19 0
    Two Hands 1:10 0
    Two Hands 1:03 0
    One Hand 3:05 3
    One Hans 2:25 3
    One Hand 1:57 0

    The average time for two hands was around 1:10 and the average number of errors was 0

    The average time for one hand was around 2:29 and the average number of errors was 2

  • Conclusions:

    • By putting the select button across the keyboard and far away from awsd, the time and errors significantly increased from 2 hands to 1 hand. If I were designing an interface that would be optimal for someone with 1 hand, I would put the select button closer to the directional keys.
